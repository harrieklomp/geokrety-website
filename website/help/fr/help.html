À propos de GeoKrety.org

GeoKrety.org (du grec geo = ‘terre’ et du polonais krety = ‘taupe’) est un service, similaire à TravelBug(TM) ou GeoLutins, permettant de suivre des objets voyageurs que vous laissez dans des géocaches. Les gens déplacent ces objets enregistrés (appelés GeoKrets) d'une cache à l'autre et signalent chaque mouvement sur ce site Web. Le trajet de chaque GeoKret peut alors être suivi sur une carte.


Une taupe

Comment prononcer 'geokrety' ? ;)

Les GeoKrets ne sont pas des objets d'échange. Vous n'êtes pas obligé de déposer un objet dans une cache pour prendre un GeoKret. Si vous voulez prendre quelque chose dans une cache alors faites un échange puis déposez ou prenez un GeoKret.

Ce service est totalement gratuit (contrairement aux TravelBug™) et facile d'utilisation. Un GeoKret peut être n'importe quel objet, mais le plus souvent il s'agit d'une petite figurine.

Créer votre propre Geokret

Regardez une vidéo sur la création de votre propre Geokret.

  1. Ouvrez un compte sur GeoKrety.org
  2. Enregistrez le GeoKret et récupérez son code de suivi unique généré automatiquement par le système (par exemple : XF3ACS). Ce code doit rester secret et ne doit être connu que de vous et de ceux qui trouvent le GeoKret
  3. Imprimez sur une étiquette obligatoirement : le nom du GeoKret, son numéro d'identification (par exemple : GK003F), son code de suivi et l'adresse du site GeoKrety.org. Vous pouvez éventuellement ajouter un objectif, votre pseudo en tant que propriétaire et quelques instructions concernant le signalement sur GeoKrety.org. Enfin, attachez l'étiquette à l'objet.

    Exemples de GeoKrets prêts au départ !

Maintenant vous pouvez :

  1. Déposer votre GeoKret dans une cache et le signaler sur ce site
  2. Prendre du plaisir en suivant le voyage de votre GeoKret !
Les différentes catégories de GeoKrets
  • Traditionnel - Une petite peluche avec son étiquette.
  • Un livre / CD / DVD ... - Si vous souhaitez suivre des livres, des CD ou des DVD déposés dans des caches, vous pouvez juste coller l'étiquette avec le numéro d'identification du GeoKret et quelques instructions au dos de l'objet.
  • Un humain - Si vous souhaitez suivre vos déplacements. Vous n'êtes pas obligés de rester dans les caches ;-).
  • Une pièce de monnaie (en anglais geocoin, en polonais toczygrosze) - Une pièce sur laquelle vous avez gravé ou ajouté son numéro de suivi.
  • KretyPost - Une carte postale.

    Principe de fonctionnement :

    1. Prenez une carte (carte postale traditionnelle ou carte de votre conception) et écrivez quelques lignes à l'attention du destinataire.
    2. Sur la carte précisez l'adresse de destination sous la forme : pseudo du destinataire, destination (continent, pays, région, ville) et une cache cible (par ses coordonnées GPS ou mieux par un WayPoint valide comme un code GeoCaching, un code OpenCaching, une ciste ...).
    3. Sur GeoKrety.org, créez un GeoKret du type "KretyPost", imprimez un timbre avec le code de suivi et collez-le sur la carte (voir quelques exemples de timbres ici).
    4. (option) Plastifiez la carte pour la protéger durant son voyage.
    5. Déposez la carte dans une cache proche de votre domicile, revenez sur GeoKrety.org pour signaler ce GeoKret comme 'déposé'.
    6. Prévenez le destinaire pour qu'il puisse suivre la progression de la carte et surtout afin qu'il récupère la carte lorsqu'elle sera arrivée dans la cache indiquée comme destination.

Les types de signalements
Icônes Description
ou J'ai déposé un GeoKret : lorsque vous ajoutez un GeoKret dans une cache.
ou J'ai pris un GeoKret : lorsque vous prenez un GeoKret dans une cache afin de le déposer rapidement dans une autre cache.
ou J'ai vu un GeoKret : lorsque vous découvrez un GeoKret et que vous le laissez dans la cache.
ou Un GeoKret a visité la cache : lorsqu'un GeoKret est en votre possession ; que vous trouvez une cache mais que vous n'y déposez pas le GeoKret. Ce type de signalement permet de faire voyager un GeoKret sans s'en séparer.
Commentaire : lorsque vous souhaitez laisser un message sur un GeoKret.
Hibernation / archive : lorsqu'un GeoKret n'a pas été vu depuis longtemps et que vous souhaitez le signaler comme perdu.

Les formats de coordonnées géographiques acceptables

Vous pouvez entrer les coordonnées lat/lon dans différents formats :

  • Degrés (DD.dddddd°)
    • 	52.205205 21.190891
	52.205205/21.190891
	N 52.205205 E 21.190891
	+52.205205 -21.190891
  • Degrés, minutes (DD°MM.mmm’)
    • 	52 12.3123 -21 11.45345
	N52 12.3123 W21 11.45345
	+52 12.3123 -21 11.45345
	N 52° 10.369' W 021° 01.542'
  • Degrés, minutes, secondes (DD°MM’SS.s”)
    • 	52 12 18.74 -21 11 27.21
	N52° 12' 18.74", W21° 11' 27.21"
	52° 12' 18.74", -21° 11' 27.21"

Numéro de référence et Code de suivi
  • Numéro de référence par exemple: GK003F. C'est le numéro publique qui est utilisé pour faire référence à un GeoKret. Il est constitué du préfixe 'GK' et de 4 digits en système hexadecimal. Vous pouvez l'utiliser pour rechercher un GeoKret ou créer un lien vers ce GeoKret. Il est visible par tout le monde.
  • Code de suivi par exemple: XF3ACS. C'est un code d'identification secret indispensable pour enregistrer la "prise", le "dépôt", ou autres actions sur un GeoKret. Seul le propriétaire et les géocacheurs qui l'ont trouvé doivent le connaitre. Il n'est pas recommandé de le divulguer (y compris sur les photos du Geokret). Bien sûr, il doit être reporté sur l'étiquette du GeoKret afin de permettre son suivi.
Altitude et code des pays

SRTM et GTOPO30 fournissent des données d'élévation et permettent la conversion de coordonnées en code pays (geonames). Ainsi à chaque étape du voyage d'un GeoKret sont assignés une altitude et un code pays (son drapeau). Grâce à ces informations, sur la page de chaque GeoKret, un profil d'altitude est dessiné pour l'ensemble de son voyage. Ces informations sont calculées toutes les 10 à 15 minutes.

QR Code

Afin de faciliter les différentes actions sur le GeoKrety lors de son voyage, j'ai ajouté la possibilité d'ajouter un QR Code sur son étiquette. Le QR Code est une sorte de code-barre qui peut être "scanné" par un smartphone compatible. Ce "scan" redirigera directement sur la page du GeoKret sur geokrety.org. Les différents codes sont encryptés dans le QR Code (en savoir plus).

Un QR code encrypté avec l'addresse de la page geokrety.org.

quand vous trouvez un GeoKret avec une étiquette contenant un QR Code vous pouvez :

  1. Scanner le QR Code
  2. Décoder et cliquer sur le lien qui s'affiche
  3. Enregistrer votre action sur le GeoKret. Le code de suivi sera indiquez par l'URL.

Comment décoder ?

Vous pouvez utiliser des logiciels libres et compatibles avec votre smartphone : i-nigma ou Kaywa Reader.

Protection des données personnelles

Votre adresse email n'est transmise à aucun tiers. Votre mot de passe est encrypté à sens unique, il ne peut donc être révélé.
